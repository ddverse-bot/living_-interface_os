# Living Interface OS

A self-aware, behavior-driven frontend interface inspired by operating systems.

This project explores what happens when UI reacts to **human behavior** instead of clicks.

## Features
- Cursor gravity & magnetic UI
- Inertia-based draggable windows
- Hesitation-aware visual states
- Idle self-rearranging layout
- Scroll-driven time-of-day shifts
- WebGL ambient background (Three.js)
- CSS Houdini procedural textures

## Tech Stack
- HTML / CSS / Vanilla JS
- GSAP
- Lenis Scroll
- Three.js
- CSS Houdini (Paint API)

## Philosophy
> Stop designing screens. Start designing behavior.
